Пошаговая инструкция: как собрать данные из WordStat

Подготовка

1

Войдите в Яндекс

Откройте wordstat.yandex.ru и авторизуйтесь в аккаунте Яндекс. Если нет аккаунта — создайте бесплатно.

2

Выберите регион

Нажмите "Все регионы" и выберите "Москва" или "Москва и область". Это критично для локального бизнеса.

3

Выберите тип данных

По умолчанию показывается "По словам" — оставьте этот режим для анализа частотности.

Сбор данных

4

Введите базовый запрос

Начните с VR арена — это покажет все запросы, содержащие эти слова.

5

Используйте операторы для точности

Оператор Пример Что показывает
без операторов VR арена Все запросы с этими словами
"кавычки" "VR арена" Только эта фраза (с разными окончаниями)
!восклицательный "!VR !арена" Точная форма слов
6

Запишите частотность

Для каждого запроса запишите число показов в месяц. Это базовая частотность.

7

Проверьте историю запросов

Переключитесь на вкладку "История запросов" для анализа сезонности и трендов.

Анализ результатов

8

Сгруппируйте запросы

Разделите запросы по категориям: прямые, услуговые, геолокационные, информационные.

9

Оцените конкуренцию

Высокочастотные запросы (>1000/мес) обычно дороже в рекламе. Низкочастотные (<500/мес) — дешевле и конверсионнее.

10

Выявите минус-слова

Найдите нерелевантные запросы (например, "VR очки купить", "VR арена работа") и добавьте их в минус-слова для рекламы.

Расчёт потенциала для Яндекс.Директ

Формула расчёта потенциальных клиентов

Формула:
Потенциальные клиенты = Частотность × CTR × Конверсия сайта

Пример расчёта для VR Fusion

Параметр Значение Пояснение
Частотность "VR арена Москва" ~3 000/мес Данные WordStat
CTR рекламы 5-10% Средний для Директа
Переходы на сайт 150-300/мес 3 000 × 5-10%
Конверсия сайта 3-7% В заявку/бронь
Лиды/мес 5-21 С одного запроса

Расчёт стоимости привлечения (CAC)

Параметр Значение
Средняя цена клика (CPC) 50-80 руб.
Кликов для 1 лида 15-30 кликов
Стоимость лида (CPL) 750-2 400 руб.
Вывод: При среднем чеке 2 500-3 000 руб. и CPL 750-2 400 руб. реклама в Директе окупается. При корпоративах (50 000+ руб.) окупаемость ещё выше.

Стратегия по сезонности

Пики спроса:
  • Осень (сентябрь-октябрь): Корпоративы, начало сезона
  • Зима (декабрь-январь): Новогодние праздники, каникулы
  • Весна (март-май): 8 марта, дни рождения, выпускные
Спад: Лето (июнь-август) — снизьте бюджет на 20-30%
