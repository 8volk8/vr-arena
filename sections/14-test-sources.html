Источник 1: Перехват лидов конкурентов

Что это такое

Сбор номеров телефонов людей, которые звонили вашим конкурентам (другим VR-клубам, квестам, развлекательным центрам). Эти люди уже интересовались подобными услугами — это тёплые лиды.

Правовой аспект: Номера телефонов сами по себе не являются персональными данными согласно 152-ФЗ. Однако при прозвоне необходимо соблюдать законодательство о рекламе (получать согласие на рекламные звонки).

Как это работает

  1. Сервис собирает номера звонивших на указанные вами номера конкурентов
  2. Вы получаете базу "тёплых" контактов
  3. Колл-центр прозванивает базу с вашим предложением
  4. Заинтересованные лиды передаются вам

Скрипт для колл-центра

Вводная часть

"Скажите, вы рассматривали VR для себя, для ребёнка или для корпоративного мероприятия?" [Ответ определяет дальнейший скрипт]

"Отлично! У нас в VR Fusion есть специальные программы для детских праздников. - Пакет на день рождения от 15 000 рублей - Включает 2 часа игры, аниматора и фотозону - Находимся в ТЦ Европолис Ростокино Могу забронировать для вас удобную дату?"

"У нас более 50 игр: шутеры, хорроры, приключения. Оборудование топового уровня — без проводов, полная свобода движений. Сейчас действует акция: первый час со скидкой 20%. Когда вам удобно прийти попробовать?"

Для B2B

"Мы проводим корпоративные мероприятия и тимбилдинги в формате VR. - Командные игры на 6-12 человек - Турниры с призами - Полное закрытие арены Могу отправить коммерческое предложение на почту?"

"Отлично, записал(а). Наш менеджер свяжется с вами для подтверждения. Напомню: VR Fusion, ТЦ Европолис Ростокино. Спасибо за ваше время, хорошего дня!"

Пошаговый запуск теста

1

Соберите список конкурентов

Найдите 10-20 номеров телефонов VR-клубов и смежных развлечений в Москве.

2

Выберите сервис перехвата

Зарегистрируйтесь на LeadFixer или LEADS SOLVER. Загрузите список номеров конкурентов.

3

Дождитесь сбора базы

Обычно занимает 3-7 дней для набора 1000 контактов.

4

Подготовьте колл-центр

Передайте скрипт, обучите операторов. Можно использовать встроенный колл-центр сервиса.

5

Настройте CRM

Создайте отдельную воронку "Перехват лидов" для отслеживания конверсии.

6

Запустите прозвон

Оптимальное время: будни 10:00-13:00 и 15:00-19:00.

7

Анализируйте результаты

Через 2 недели оцените: CPL, конверсию, качество лидов. Решите о масштабировании.

Источник 2: Яндекс.Бизнес (Рекламная подписка на 180 дней)

Что это такое

Яндекс.Бизнес — платная подписка для продвижения в сервисах Яндекса: Карты, Поиск, Директ, Дзен. Одна подписка = показы во всех сервисах.

Что входит в подписку:
  • Приоритетное размещение в Яндекс.Картах
  • Реклама в поисковой выдаче Яндекса
  • Показы в Яндекс.Директ (автоматические)
  • Продвижение в Яндекс.Дзен
  • Бесплатный подменный номер для звонков

Проблема: сложности сбора метрик

Ограничения Яндекс.Бизнес:
  • Нет API для автоматической выгрузки данных
  • Ограниченная статистика в кабинете
  • Нет прямой интеграции со сквозной аналитикой
  • Сложно отследить полный путь клиента
  • Нельзя разделить трафик по сервисам (Карты vs Поиск)

Решения для сбора метрик с Яндекс.Бизнес

Решение 1: UTM-метки

1

Добавьте UTM к ссылке на сайт

В настройках Яндекс.Бизнес укажите ссылку на сайт с UTM-метками:

https://vrfusion.ru/?utm_source=yandex_business&utm_medium=maps&utm_campaign=subscription_180

Решение 2: Отдельный номер телефона

2

Используйте подменный номер

Яндекс.Бизнес предоставляет бесплатный подменный номер. Все звонки на него = трафик с Яндекс.Бизнес.

Альтернатива: подключите коллтрекинг (Calltouch, Callibri) и создайте статический номер для Яндекс.Бизнес.

Решение 3: Уникальные промокоды

3

Создайте промокод для Яндекс.Бизнес

Например: YANDEX10 — скидка 10% для клиентов, пришедших с Яндекс.Карт.

Укажите промокод в описании карточки на Яндекс.Картах.

Решение 4: Ручной учёт

4

Спрашивайте клиентов

При бронировании/звонке спрашивайте: "Откуда вы о нас узнали?"

Фиксируйте ответ в CRM в поле "Источник".

Настройка отслеживания Яндекс.Бизнес

1

Включите подменный номер

В кабинете Яндекс.Бизнес → Настройки → Подменный номер → Включить.

2

Добавьте ссылку с UTM

Яндекс.Бизнес → Редактировать → Сайт → укажите URL с UTM-метками.

3

Установите Яндекс.Метрику на сайт

Если ещё не установлена — установите. Создайте цели для отслеживания конверсий.

4

Создайте отчёт в Метрике

Отчёты → Источники → Метки UTM → фильтр по utm_source=yandex_business.

5

Настройте еженедельный экспорт

Раз в неделю выгружайте данные из кабинета Яндекс.Бизнес вручную и сопоставляйте с Метрикой.
